# My CS50P Journey

## Overview

This repository contains my final, polished solutions for **Harvard's CS50P** (*Introduction to Programming with Python*).

Rather than including every early draft or original iteration, this codebase strictly features my **latest and cleaned-up submissions** for each exercise, formatted according to clean code best practices.

---

## Repository Structure & Highlights

* **Course:** Harvard CS50P (*CS50’s Introduction to Programming with Python*)
* **Code Standard:** Final submissions with cleaned formatting, removed top-level header noise, and standardized Python style guidelines.
* **Content:** Selected problem sets and project submissions reflecting my complete learning path through the course.

---

## Tech & Tools

* **Language:** `Python 3`
* **Environment:** CS50 DevSpace / VS Code
* **Key Topics Covered:** Functions, Variables, Conditionals, Loops, Exceptions, Libraries, Unit Tests, File I/O, Regular Expressions, and Object-Oriented Programming (OOP).
* **Libraries:** `random`, `math`, `sys`, `re`, `argparse`, `requests`, `csv`, `pytest`, `pillow`, `fpdf2`, `emoji`, `pyfiglet`, `inflect`, `customtkinter` (final project), `pycountry` (final project).

## Repository Structure

Each week has separate folder for its problems. And for each problem there is separate folder for it in
its respective week's folder. The problem folders may comprise of a single file (eg. Problem: Indoor Voices)
as the solution, or may comprise of multiple files. The problem folders may comprise of subfolders too.

```text
[My_CS50p_journey Repository]
├── week0/      # Functions, Variables
│   ├── indoor/         # Problem: Indoor Voices
│       └── indoor.py
│   ├── playback/       # Problem: Playback Speed
|       └── playback.py
│   ├── faces/          # Problem: Making Faces
|       └── faces.py
│   └── ...
├── week1/      # Conditionals
│   └── ...
├── week2/      # Loops
│   └── ...
├── week3/      # Exceptions
│   └── ...
├── week4/      # Libraries
│   └── ...
├── .../
├── README.md
└── LICENSE
```
